Saves a completed purity test result to MongoDB.

## Request body

<ParamField path="body.score" type="number" required>
  Purity score from 0 to 100.
</ParamField>

<ParamField path="body.answers" type="boolean[]" required>
  Array of 100 boolean values representing each question answer. `true` indicates the user answered "yes" to that question.
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  `true` if the result was saved successfully.
</ResponseField>

### Error response

Status: `500`

<ResponseField name="error" type="string">
  Error message: `"Failed to save test result"`
</ResponseField>
