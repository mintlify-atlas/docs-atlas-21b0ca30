# Introduction

Welcome to the SJSU Purity Test documentation! The SJSU Purity Test is a 100-question campus experience tracker designed specifically for San Jose State University students. It's a fun way for Spartans to reflect on their college journey, compare experiences with peers, and discover how they stack up against the community.

## What is the SJSU Purity Test?

The SJSU Purity Test presents students with 100 questions covering a wide range of campus experiences—from academic moments like "Pulled an all-nighter to finish a project or study for an exam?" to social experiences like "Been to a frat party?" and unique SJSU situations like "Was in the SU during the Panda Express fire?" or "Got rejected from every UC?"

Students check off experiences they've had, and at the end receive a **purity score** calculated using the formula:

```typescript
const score = Math.round(((questions.length - checkedCount) / questions.length) * 100);
```

A higher score means more "pure"—fewer experiences checked off. A lower score means you've been around the block at SJSU!

## Tech Stack

The SJSU Purity Test is built with modern web technologies:

- **Next.js 15** with App Router and Turbopack
- **React 19** for the UI
- **TypeScript** for type safety
- **MongoDB** with Mongoose for data persistence
- **Tailwind CSS 4** for styling

The app features a clean, vintage-inspired design with a cream and dark red color scheme that gives it a classic college feel.
