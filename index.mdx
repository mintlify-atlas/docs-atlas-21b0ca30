## Welcome to the SJSU Purity Test

The SJSU Purity Test is a 100-question, campus-specific experience tracker inspired by the classic "rice purity test" format — reimagined for San Jose State University students. It lets you and your fellow Spartans check off real campus experiences, calculate a purity score, and see how you compare against the community.

## About this project

Built with **Next.js 15**, **React 19**, **TypeScript**, and **MongoDB** (via Mongoose), the SJSU Purity Test stores every submitted result and aggregates community-wide statistics in real time. The entire project was prototyped in a couple of hours using Cursor.

<Note>
  This is not a bucket list. As the app itself warns: you are beyond cooked if you complete all 100 items.
</Note>
